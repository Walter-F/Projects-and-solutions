При выполнении тестового задания использовалась локальная база данных, прикрепляемая к проекту, СУБД phpMyAdmin, комплекс софта MAMP для запуска локального сервера.
Ручной импорт данных из таблицы Excel производился путём преобразования файла в .CSV формат и последующая загрузка на сервер. 
Ссылка для установки комплекса MAMP - https://www.mamp.info/en/downloads/
